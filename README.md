## Important honesty note

**No contact details were fabricated.** Every phone number, name, and address on the
site was actually visible on a live public web page, and each section links to its
source. Where a number was not publicly visible (e.g. IndiaMART listings that gate the
number behind an enquiry), it is clearly marked **"hidden"** rather than guessed —
because you'll be calling these sellers with real money, fake data would be worse than none.

Prices and stock on premium monitors change weekly. **Always re-confirm the price,
stock, genuine `-W` model, and GST invoice directly with the seller before paying.**
