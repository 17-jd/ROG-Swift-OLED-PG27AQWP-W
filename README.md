an IT buyer in **Surat, Gujarat** — verified specs, India pricing, and **real dealer
contacts** for sourcing in bulk:

1. **ROG Swift OLED PG27AQWP-W** — 26.5" QHD 540 Hz Tandem WOLED gaming monitor (White)

## How to open

Open **`index.html`** in any web browser (double-click it, or drag it into a browser tab).
Use the top bar to switch between **Home → Monitor → Wi-Fi 7 Router**. No build step, no
server, no dependencies — plain static HTML + one CSS file.

```
index.html      → landing page + quick comparison of both products
monitor.html    → full guide for the PG27AQWP-W monitor
router.html     → full guide for the GT-BE19000AI router
styles.css      → shared styling
```

Each product page has the same sections: Overview, Full Specs, Pricing, Dealers & Contacts,
Surat/Gujarat sourcing, Buying Advice, Reviews, and a "What we couldn't verify" transparency
section. Every section links to its source.

## Important honesty note

**No contact details were fabricated.** Every phone number, name, and address was visible on
a real public web page, and each section links to its source. Three honesty conventions are
used throughout:

- **"Phone public"** — number was visible on the seller's own site.
- **"Phone hidden"** — IndiaMART/TradeIndia gates the number behind a free enquiry; we show
  the company + city + listing URL, never a guessed number.
- **"Number unverified"** — appeared only in a third-party directory (e.g. the Surat ASUS
  dealers); shown as a *lead to verify*, clearly labelled, because we could not confirm the
  digits ourselves.

Prices, stock, and availability on premium hardware change weekly. **Always re-confirm the
price, stock, exact SKU, and GST invoice directly with the seller before paying.**
